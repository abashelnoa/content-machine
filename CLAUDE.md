# הוראות לשימוש ב-Claude בפרויקט זה

## חיפוש מידע עדכני — חובה
לפני כל תשובה הקשורה ל-API, כלים, מודלים, או best practices — השתמש ב-**WebSearch** כדי לאמת שהמידע עדכני.

המידע הפנימי של Claude עלול להיות ישן. תמיד בדוק תיעוד עדכני לפני שאתה כותב קוד שמשתמש ב:
- endpoints של fal.ai (שמות מודלים, פרמטרים)
- גרסאות ו-API של Anthropic/Claude
- חבילות Python ו-Streamlit
- כל שירות חיצוני אחר

## הקשר הפרויקט
מערכת לייצור פוסטים בעברית בסגנון "לירז" + תמונות תואמות עם זהות המשתמש.
- טקסט: Claude API
- תמונות: fal.ai (InstantID לשמירת זהות)
- ממשק: Streamlit
